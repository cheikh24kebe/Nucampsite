# Nucampsite

Nucampsite is a fictional campsite review website built as part of my JavaScript bootcamp course. It features multiple pages, styled with SCSS and Bootstrap, and is bundled with Parcel.js.

## 🚀 Features

- Multi-page website (`index.html`, `aboutus.html`, `contactus.html`)
- Responsive layout with Bootstrap
- SCSS styling and modular CSS
- Parcel bundler for easy development
- Font Awesome integration (icons)

## 🛠 Technologies

- HTML5  
- SCSS  
- Bootstrap 5  
- Font Awesome  
- JavaScript (ES6)  
- Parcel.js  
- Git & GitHub
